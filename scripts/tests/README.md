# scripts/tests

データ処理スクリプトの検証テストを格納するディレクトリです。

## テストの実行

```bash
cd scripts
uv sync --group dev  # 初回のみ
uv run pytest tests/ -v
```

## テストファイル

| ファイル                  | 説明                                                                     |
| ------------------------- | ------------------------------------------------------------------------ |
| `test_data_assignment.py` | 地価・課税所得・気象データが正しい市区町村に割り当てられていることを検証 |
