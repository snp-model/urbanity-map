# Urbanity Map

全国市町村の「都会度」を可視化するインタラクティブマップアプリケーション。

## 概要

Urbanity Map は、日本全国の市町村ごとに「都会度」スコアを算出し、地図上で視覚的に表示する Web アプリケーションです。移住検討者や一般ユーザーが、各地域の都市機能の充実度を直感的に把握できます。

## プロジェクト構成

```
urbanity-map/
├── docs/                    # ドキュメント
│   ├── 01_調査/             # 調査資料
│   ├── 02_要件定義/         # 要件定義書・ADR
│   └── 03_設計/             # システム設計書
├── frontend/                # フロントエンドアプリケーション (React + Vite)
└── README.md
```

## 技術スタック

- **フロントエンド**: React, TypeScript, Vite
- **地図ライブラリ**: MapLibre GL JS
- **タイルサーバー**: 国土地理院（淡色地図）
- **スタイリング**: Vanilla CSS

## クイックスタート

```bash
# フロントエンドのセットアップ
cd frontend
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。
